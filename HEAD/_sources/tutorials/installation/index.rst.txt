Installation
===================

tprofiler is currently hosted on PyPI. It required python >= 3.8.

You can simply install tprofiler from PyPI with the following command:

.. code:: shell

    pip install tprofiler

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/hansbug/tprofiler@main

script:
